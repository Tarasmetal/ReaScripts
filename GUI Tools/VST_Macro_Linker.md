## 📝 Описание

**VST Macro Linker** - это мощный инструмент для REAPER DAW, который позволяет связывать параметры VST плагинов с цветными макро-контроллерами. Скрипт предоставляет визуальный интерфейс с 16 макро-слайдерами и 16 падами для управления параметрами эффектов в реальном времени.

### Ключевые особенности:
- **16 цветных макро-слайдеров** для плавного контроля параметров
- **16 цветных падов** для переключения состояний (вкл/выкл)
- **MIDI Learn** функция для быстрого маппинга контроллеров
- **Автоматизация** через специальный JSFX backend
- **Визуальная обратная связь** с цветовой кодировкой
- **Сохранение пресетов** для быстрой смены конфигураций

## ✨ Основные возможности

### 🎚️ Макро-слайдеры (16 штук)
- Плавное управление параметрами от 0% до 100%
- Уникальная цветовая кодировка для каждого макро
- Возможность инверсии диапазона
- Настройка Min/Max значений
- Offset параметр для смещения модуляции

### 🔲 Пады (16 штук)
- Мгновенное переключение состояний (вкл/выкл)
- Идеально подходят для bypass эффектов
- Визуальная индикация состояния
- MIDI маппинг на ноты или CC

### 🎹 MIDI интеграция
- **MIDI Learn** - просто нажмите кнопку и двигайте контроллер
- Поддержка CC и Note сообщений
- Сохранение MIDI маппингов в пресеты
- Визуальный индикатор MIDI активности

### 🔄 Автоматизация
- Полная интеграция с системой автоматизации REAPER
- 6 режимов автоматизации:
  - **Trim/Read** - чтение с возможностью временной корректировки
  - **Read** - только чтение автоматизации
  - **Touch** - запись при касании
  - **Latch** - запись после касания до остановки
  - **Latch Preview** - предпросмотр с защёлкой
  - **Write** - постоянная запись

## 💾 Установка

### Требования:
- **REAPER** версии 7.03 или выше (для MIDI Learn)
- **ReaImGui** библиотека
- **JS Extension** (опционально, для расширенных функций)
- **SWS Extension** (опционально, для улучшенной работы с треками)

### Процесс установки:

1. **Установка основного скрипта:**
   ```
   Поместите файл trs_VST_Macro_Linker.lua в папку Scripts
   ```

2. **Установка JSFX backend (для автоматизации):**
   ```
   Файл trs_VST_Macro_Linker.jsfx должен находиться в той же папке
   При первом запуске скрипт попытается установить его автоматически
   ```

3. **Загрузка в REAPER:**
   - Откройте Actions List (?)
   - Нажмите "Load ReaScript..."
   - Выберите trs_VST_Macro_Linker.lua
   - Запустите скрипт

## 🖥️ Интерфейс

### Верхняя панель
```
[✓] Automation    [●]     <- MIDI индикатор
[Trim/Read] [Read] [Touch] [Latch] [Latch Prev] [Write]
```

### Секция пресетов
```
MIDI Mapping: [Выбор пресета ▼] [Save] [Save As]
```

### Макро-слайдеры
```
[M] [Link] ════════════◉══════ Macro 1: 0.75
[M] [Link] ═══◉═══════════════ Macro 2: 0.25
...
```
- **[M]** - MIDI Learn (зелёный = назначен, серый = не назначен, жёлтый = обучение)
- **[Link]** - Связать с последним тронутым параметром
- **Слайдер** - Управление значением макро

### Пады
```
[◼] [◼] [◻] [◼]     <- Визуальное состояние
[◼] [◻] [◼] [◼]        (яркий = вкл, тусклый = выкл)
```

### Linked Parameters (связанные параметры)
```
▼ Linked Parameters : 5
────────────────────────
[●] X M1 [Track 1] Reverb : Mix
    [ ] Invert  Min: 0.00  Max: 1.00  Offset: 0.50
────────────────────────
[●] X P3 [Master] Compressor : Threshold
    [ ] Invert  Min: 0.20  Max: 0.80
```

## 🎯 Использование

### Базовый рабочий процесс:

#### 1. Связывание параметров с макро:
```
1. Откройте плагин VST в REAPER
2. Кликните на параметр, который хотите контролировать
3. В VST Macro Linker нажмите кнопку [Link] нужного макро
4. Параметр теперь связан и управляется макро-слайдером
```

#### 2. MIDI маппинг:
```
1. Нажмите кнопку [M] рядом с макро (станет жёлтой)
2. Двигайте ручку на MIDI контроллере
3. Маппинг создан автоматически (кнопка станет зелёной)
```

#### 3. Настройка диапазона:
```
1. В секции Linked Parameters найдите нужный параметр
2. Настройте Min/Max для ограничения диапазона
3. Используйте Invert для реверса направления
4. Offset смещает центр модуляции
```

### Работа с падами:

#### Связывание пада с параметром:
```
1. Кликните на параметр VST (например, Bypass)
2. Ctrl+Click на нужном паде
3. Пад теперь переключает параметр между 0 и 1
```

#### MIDI маппинг пада:
```
1. Ctrl+Right Click на паде
2. Нажмите клавишу или пад на MIDI контроллере
3. Готово! Пад реагирует на MIDI
```

## 🎹 MIDI маппинг

### Поддерживаемые типы:
- **Control Change (CC)** - для слайдеров и падов
- **Note On/Off** - для падов (триггерный режим)

### Создание MIDI пресета:
```
1. Настройте все нужные MIDI маппинги
2. Нажмите [Save As] в секции MIDI Mapping
3. Введите имя пресета
4. Пресет сохранён и доступен в выпадающем списке
```

### Структура пресетов:
Пресеты сохраняются в папке `MacroPresets/` в формате:
```
macro_index,status_byte,cc_number
1,176,10    <- Макро 1 на CC#10 канал 1
17,144,60   <- Пад 1 на ноту C3
```

## 🔄 Автоматизация

### Активация автоматизации:

1. **Включите checkbox "Automation"**
   - Автоматически создаётся трек "VST Macro Linker"
   - На треке размещается JSFX эффект для записи автоматизации

2. **Выберите режим автоматизации:**
   - **Read** - воспроизведение записанной автоматизации
   - **Touch** - запись при движении слайдера
   - **Latch** - продолжает запись после отпускания
   - **Write** - перезаписывает автоматизацию

### Запись автоматизации:
```
1. Включите Automation
2. Выберите режим Touch или Write
3. Нажмите Play в REAPER
4. Двигайте макро-слайдеры
5. Автоматизация записывается на трек
```

## ⌨️ Горячие клавиши

### Глобальные (требуют JS Extension):
- **`1`** - Связать последний тронутый параметр с последним использованным макро
- **`2`** - Связать последний тронутый параметр со следующим свободным макро

### В окне скрипта:
- **`H`** - Показать/скрыть секцию Linked Parameters
- **`Esc`** - Закрыть окно скрипта

### Мышь:
- **Left Click** на [Link] - связать параметр
- **Right Click** на [Link] - удалить все связи макро
- **Ctrl + Left Click** на Pad - связать параметр с падом
- **Ctrl + Right Click** на Pad - MIDI Learn для пада
- **Right Click** на Pad - удалить связи пада

## 💾 Сохранение и загрузка

### Автосохранение:
Скрипт автоматически сохраняет:
- Все связанные параметры
- Имена макро
- Настройки Min/Max/Offset
- Состояние Automation

Файл: `ProjectName_ML.ini` в папке проекта

### Ручное сохранение:
```
[L] - Load - загрузить конфигурацию из файла
[S] - Save - сохранить конфигурацию в новый файл
```

### Структура файла настроек:
```ini
[Version]
2.9

[Settings]
ShowLinkedParams=true
AutomationEnabled=true

[LinkedParams]
track_guid|fx_id|param_id|macro_idx|min|max|inverted|track_name|fx_name|param_name|offset

[MacroNames]
Bass Control
Treble Control
Reverb Mix
...
```

## 📚 Примеры использования

### Пример 1: Управление миксом эффектов
```
1. Откройте Reverb на треке вокала
2. Кликните на параметр "Dry/Wet Mix"
3. Нажмите [Link] на Macro 1
4. Переименуйте в "Vocal Reverb"
5. Настройте Min: 0%, Max: 30% для тонкого контроля
```

### Пример 2: Bypass нескольких эффектов
```
1. Свяжите Pad 1 с Bypass компрессора
2. Свяжите Pad 2 с Bypass EQ
3. Свяжите Pad 3 с Bypass дилея
4. Теперь можете мгновенно включать/выключать эффекты
```

### Пример 3: Морфинг между пресетами
```
1. Настройте эффект в положение A
2. Свяжите параметры с макро, установите как Min значения
3. Настройте эффект в положение B
4. Установите текущие значения как Max
5. Теперь макро морфит между двумя состояниями
```

## 🔧 Решение проблем

### MIDI не работает:
```
✓ Проверьте REAPER Preferences > MIDI Devices
✓ Включите "Enable input for control messages"
✓ Убедитесь, что MIDI индикатор мигает при движении контроллера
✓ Обновите REAPER до версии 7.03+
```

### JSFX Missing ошибка:
```
✓ Нажмите кнопку [Reload]
✓ Или вручную скопируйте trs_VST_Macro_Linker.jsfx в:
  REAPER/Effects/Taras Scripts/
```

### Параметры не обновляются:
```
✓ Проверьте, что трек с эффектом существует
✓ Убедитесь, что FX окно было открыто хотя бы раз
✓ Попробуйте переназначить параметр
```

### Автоматизация не записывается:
```
✓ Включите checkbox "Automation"
✓ Выберите режим Touch или Write
✓ Проверьте наличие трека "VST Macro Linker"
✓ Убедитесь, что воспроизведение запущено
```
