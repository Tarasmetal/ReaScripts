## **📖 Описание**
**Move selected items files to region folder** — это специализированный инструмент для организации медиа-файлов вашего проекта. Скрипт позволяет автоматически перемещать исходные файлы выбранных айтемов в подпапки, названные в честь регионов, в которых эти айтемы находятся.

Это идеальное решение для структурирования данных при записи дублей, работе с библиотеками звуков или подготовке ассетов для игр, когда файлы должны быть разложены по категориям/папкам.

---

## **✨ Ключевые возможности**
- **Автоматическая сортировка:** Файлы физически перемещаются в подпапки внутри директории вашего проекта.
- **Два режима детекции:** 
    - По позиции айтема (каждый айтем в свою папку региона).
    - По позиции курсора (все айтемы в папку региона под курсором).
- **Безопасный предпросмотр:** Таблица показывает исходные имена и будущие пути до выполнения перемещения.
- **Умная очистка имен:** Автоматически заменяет недопустимые символы в именах регионов на безопасные подчеркивания.
- **Поддержка SWS и Shell:** Несколько уровней проверки для надежного перемещения файлов в Windows и macOS.
- **Автоматическое обновление путей:** REAPER мгновенно узнает о новом местоположении файлов без потери связи (Media Offline -> Online).

---

## **🛠️ Как это работает**
1. Скрипт анализирует выбранные айтемы и определяет их исходные файлы на диске.
2. Проверяет наличие регионов в указанной позиции (курсор или старт айтема).
3. Генерирует новый путь: `Путь_Проекта / Имя_Региона / Имя_Файла`.
4. Временно переводит айтемы в состояние **Offline**, чтобы разблокировать доступ к файлам.
5. Перемещает файлы физически и обновляет ссылки внутри проекта на новые места.
6. Возвращает айтемы в состояние **Online**.

## **🚀 Использование**

### **Шаг 1: Выбор айтемов**
Выделите один или несколько айтемов в окне аранжировки, файлы которых вы хотите переместить.

### **Шаг 2: Настройка режима**
В окне скрипта выберите подходящий режим:
- **Галочка "Use Edit Cursor Position" включена:** Скрипт найдет регион, в котором стоит ваш курсор редактирования, и предложит переместить все выбранные файлы именно в папку этого региона.
- **Галочка выключена (по умолчанию):** Скрипт для каждого айтема индивидуально определит регион, в котором он начинается.

### **Шаг 3: Сканирование**
Нажмите кнопку **"Scan Selected Items"**. В таблице появится список файлов, готовых к перемещению, и их будущие пути.

### **Шаг 4: Перемещение**
Если всё выглядит корректно, нажмите кнопку **"MOVE FILES"**. Скрипт выполнит операцию и обновит статус.
